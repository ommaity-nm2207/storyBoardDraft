Background and Inspiration

I've been interested in Video Games for a really long time. As of lately, I've been extremely interested in interactive games because I've enjoyed the idea of the story shaping according to the decisions you make. I've been quite excited at the prospect of making my own small video game with some witty dialogue and allowing the user to try and explore all the different options presented to them.

For inspiration, I've taken some ideas from a 2 games.

    Emily is away
    Sara is Missing

Objectives

I would like to make an interactive short story(more like a short teaser), with an additional maze like puzzle added to the game to allow the user to perform some problem solving. There will be a few alternative pathways for the story to progress (currently looking at 2). This will be the core and primary objective.

If time permits, I would love to implement a feature which would let the user undo their actions so that they can explore more easily. A timer could also be added to the puzzle

Data

The dialogue which the user encounters will change depending on the input. The game will keep track of the sequence of inputs to change dialogue appropriately.

The maze will require a use of a 2d array to create a list of possible moves. Functions will be added to keep track of the users progress through the maze, as well as determine any conditions for game over (e.g hit the wall)

Since the game will require about 5 minutes per run, it might get frustrating to start the game from the beginning every time the page reloads. If time persists, I would like to implement a feature where the user can continue from where they left off (need to consult TA for what to learn for this feature)
